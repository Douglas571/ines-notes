diagrama entidad relación del proyecto

problema: El departamento de planificación y evaluación curricular de la UNEFA, necesita un sistema informático que le permita llevar de forma digitalizada el proceso de seguimiento y evaluación docente.

El sistema debe poder facilitar sus tareas cotidianas; la base de datos debe poder almacenar datos de:

  - usuario
    - correo
    - contraseña
    - tipo

    - nombres
    - apellidos
    - cedula

  - profesores:
    - usuario_id

  - aulario
    - module
    - Aula

    - materia_id
    - dia
    - hora_inicio
    - hora_fin

  - materias
    - id
    - carrera
    - profesor_id

  - evaluadores
    - usuario_id
    - es_coordinador
    - carrera_asignada

  - resultados_evaluación_por_estudiantes
    - id
    - fecha
    - evaluador_id
    - profesor_id

    - instrumento_id
    - resultados (json)

  - resultados_evaluación_por_coordinador
    - id
    - fecha
    - evaluador_id
    - profesor_id

    - instrumento_id
    - resultados (json)

  - resultados_evaluación_por_acompañamiento
    - id
    - fecha
    - evaluador_id
    - profesor_id

    - instrumento_id
    - resultados (json)

  - Instrumentos (formularios)
    - id
    - preguntas (JSON)
